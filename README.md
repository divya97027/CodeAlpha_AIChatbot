# AI Chatbot (Java)

This project is a Java-based AI chatbot developed using rule-based NLP techniques and a Swing GUI.

## Features
- Interactive chatbot
- Rule-based NLP responses
- FAQ trained responses
- GUI for real-time interaction

## Technologies Used
- Java
- Swing
- NLP (keyword matching)

## How to Run
javac AIChatbotGUI.java
java AIChatbotGUI
